Welcome

This book represents an collection of data reports related to the study: Does Sleep Regularity Worsen Over the Third Trimester of Pregnancy?.

All analyses presented in this document are fully reproducible and were conducted using the Quarto publishing system and the R programming language. Click here or on the GitHub icon in the menu to explore the code. The research compendium is also accessible by clicking here via The Open Science Framework.
